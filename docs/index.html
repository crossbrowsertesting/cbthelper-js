cbthelper (for javascript)

cbthelper wraps CrossBrowserTesting's selenium api into an easy to use library.

Install

npm install --save cbthelper

Usage

See example.js to see how you can use the library. Don't forget to set your username and authkey in that file. You can also set the environment variables CBT_USERNAME and CBT_AUTHKEY.
